# AI Agent Ecosystem — Interactive Dashboard

A React + Vite web prototype that visualizes the relationships between **AI Agents**, **Foundation Models**, and **Tools**.

## Features

- **Agents** — Browse 6 specialized agents (Research, Coding, Data, Creative, DevOps, QA)
- **Models** — Explore foundation models (GPT-4o, Claude, Codex, Gemini, LLaMA)
- **Tools** — Discover tools like Web Search, Code Executor, Git Operations, and more
- **Relationships** — See how agents connect to models and tools in one unified view
- Click any card for a detail panel showing specs and connections

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Tech Stack

- React 19 + Vite
- Pure CSS (no UI library)
- Dark-themed, responsive design
